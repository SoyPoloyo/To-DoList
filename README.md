# To-Do List App

Aplicacion de lista de tareas usando de base el webpack-starter

### Notas: 
Parte de un template para el html y css
```
Template creado por "http://sindresorhus.com" Sindre Sorhus
```
Primero reconstruir los modulos de Node
```
npm install
```

Y para construir el build
```
npm run build
````
